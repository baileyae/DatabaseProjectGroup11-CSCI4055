This project is a Java Swing application that connects to a database using the COMPANY schema.  
It loads departments and projects, and allows the user to search for employees based on:

- Departments (multiple selection)
- Projects (multiple selection)
- Employees **in** selected items
- Employees **NOT in** selected items
- Scrollable lists for all categories
